重新以管理员权限运行 npm install -g @angular/cli

如果报错，就先删掉 C:\Users\adminstrator\AppData\Roaming\npm\node_modules\@angular\cli\node_modules 目录，然后再运行 npm install -g @angular/cli

等安装完成后 ng version 就可以了
