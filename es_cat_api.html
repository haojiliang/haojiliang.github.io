elasticsearch cat api 查询索引信息

2019-05-09

v：verbose 有无标题
h：headers 返回的字段
s：排序
bytes：store.size 单位，数字格式输出，不带 mb，gb 等单位，方便在代码中计算
help：查看字段意思

官方文档：https://www.elastic.co/guide/en/elasticsearch/reference/current/cat-indices.html
